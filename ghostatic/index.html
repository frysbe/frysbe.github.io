Ghostatic

Static site generator for Ghost

Ghost is a pretty sweet blogging platform. But there are not many options for hosting yet. So why not use GitHub pages, a free and high performance platform for static hosting?

All you need is Ghostatic to convert your Ghost blog to a static site by a single click (or tap) on a button.

Ghostatic is open source and available under the same license as Ghost (MIT).
